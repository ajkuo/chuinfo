# chuinfo
中華大學資訊網 CHU Info
  
此網站由 php + MySQL 撰寫而成。  
Ps. 此處並未包含中華大學課程爬蟲程式。  
  
--  
  
  這是一個 for 中華大學學生的資訊網，主要可以對課程進行評價、留言分享，並設有討論區可以自由發表主題。
